# homebrew-applefm

Homebrew tap for [applefm](https://github.com/yoshikouki/applefm) - a thin CLI wrapper for Apple's Foundation Models framework.

## Installation

```bash
brew tap yoshikouki/applefm
brew install applefm
```

## Requirements

- macOS 26+ (Tahoe)
- Apple Silicon Mac with Apple Intelligence enabled

## Usage

```bash
applefm model availability   # Check model availability
applefm respond "Hello"       # One-shot generation
applefm --help                # See all commands
```

For full documentation, see the [applefm repository](https://github.com/yoshikouki/applefm).
